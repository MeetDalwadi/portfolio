Weather App

A modern and responsive weather application that provides real-time weather information for any location worldwide.

Key Features

  • Real-time weather data display
  • Global city search functionality
  • Detailed weather conditions
  • Temperature and humidity data

Technical Features

  • Responsive design
  • API integration
  • Clean user interface
  • Cross-browser compatibility

Technologies Used

HTML5 CSS3 JavaScript OpenWeatherMap API

Setup Instructions

  • Clone the repository:
    git clone https://github.com/MeetDalwadi/weather-app.git
  • Navigate to project directory:
    cd weather-app
  • Open in browser:
    Open index.html in your preferred web browser
  • API Setup:
    Sign up at OpenWeatherMap and replace the API key in script.js
