Authentication Login System

A secure Java web application built with Jakarta EE that implements user authentication with email verification.

Key Features

  • User email verification system
  • Secure token-based verification
  • SMTP email integration with Gmail
  • Real-time active user tracking
  • Admin dashboard with user analytics

Security Features

  • Secure random token generation
  • SQL injection prevention
  • TLS-enabled SMTP connection
  • Prepared statements for database queries

Technologies Used

Java Jakarta EE Maven MySQL 8.3.0 Jakarta Mail PHPMyAdmin Apache Tomcat
